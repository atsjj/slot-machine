# Slot Machine

Toy slot machine built in Rust using Relm and GTK+

## Requirements

* [GTK+](https://gtk-rs.org/docs-src/requirements)
* [Rust](https://www.rust-lang.org/tools/install)

## Compiling

* `git clone https://github.com/atsjj/slot-machine.git`
* `cargo build`
